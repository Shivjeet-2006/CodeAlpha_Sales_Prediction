# Sales Prediction using Python

## CodeAlpha Data Science Internship

### Project Overview

This project focuses on predicting product sales based on advertising expenditures across different marketing channels. Using Machine Learning techniques, the model analyzes the relationship between advertising investments and sales performance to help businesses make data-driven marketing decisions.

### Problem Statement

Businesses invest heavily in advertising through various channels such as TV, Radio, and Newspapers. The objective of this project is to predict future sales based on advertising spending and identify which advertising channels have the greatest impact on sales outcomes.

### Dataset Information

The dataset contains advertising budgets allocated to different media platforms:

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales (Target Variable)

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

### Machine Learning Algorithm

* Linear Regression

### Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Train-Test Split
6. Model Training using Linear Regression
7. Sales Prediction
8. Model Evaluation
9. Business Insights Generation

### Evaluation Metrics

The model performance was evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

### Visualizations Included

* TV Advertising vs Sales
* Radio Advertising vs Sales
* Newspaper Advertising vs Sales
* Advertising Channel Impact Analysis
* Actual vs Predicted Sales Comparison

### Key Insights

* Advertising expenditure has a significant influence on sales performance.
* TV and Radio advertising generally show stronger relationships with sales growth.
* Data-driven budget allocation can improve marketing effectiveness.
* Predictive analytics can help businesses optimize advertising investments and maximize return on investment (ROI).

### Results

The Linear Regression model successfully predicted sales using advertising expenditure data and demonstrated strong predictive performance through evaluation metrics.
